                            **SQL Exploratory Data Analysis & Advanced Analytics**
*Project Overview-
This project focuses on extracting actionable business intelligence from raw datasets using advanced SQL techniques. It covers the full lifecycle of data analysis, from initial Exploratory Data Analysis (EDA) to complex analytical reporting.

*Core Analytics Phases-
1. Exploratory Data Analysis (EDA)
  --> Database Profiling: Investigated data distribution, identified key dimensions/measures, and checked for data quality issues.
  --> Insight Discovery: Uncovered business trends, such as top-performing products and customer segments, by combining descriptive dimensions with quantitative metrics.
2. Advanced Data Analytics 
  --> Advanced SQL Techniques: Leveraged Window Functions (Aggregate, Ranking, and Value functions) to perform time-series analysis, month-over-month, and year-over-year comparisons.
  --> Complex Reporting: Developed consolidated views to answer critical business questions and generate reports for stakeholders.
Key Technical Skills
Window Functions: RANK(), DENSE_RANK(), LEAD(), LAG(), and SUM() OVER().
Advanced Querying: Utilization of Common Table Expressions (CTEs) and Subqueries to streamline complex logic.
Business Logic: Implementing segmentation, ranking, and trend analysis to drive decision-making.
